Diseñando una pagina Web solo

1. No te limites a usar herramientas visuales

Existen muchas herramientas, incluso gratuitas, que te permiten crear paginas web sin saber HTML.

Esto es una manera muy rápida de crear paginas web porque te "oculta" el código y trabajás de una manera similar a como ya estás acostumbrado a hacerlo en los editores de texto. Y realmente está muy bien para hacer, por ejemplo, una primera versión de una página web o partes de ella.

Pero no hay que dejar de aprender HTML a fondo. Esto sería pan para hoy y hambre para mañana porque así siempre tendrás un techo muy bajo, cosas un poco más avanzadas no se pueden hacer así, ojo!.

2. Pensá una buena estructura común

Antes de crear una página web, se debe pensar una estructura lógica común compuesta por etiquetas de divisiones.

Un ejemplo puede ser una estructura básica de: cabecera / cuerpo /pie.

Cuando quieras aplicar diseño con CSS a la página te resultará más fácil y verás cómo una estructura bien pensada te dará margen paara crear un buen diseño.

3. Diseñar "responsive" desde el principio

La mayor parte de visitas se hacen desde dispositivos móviles ybes cada vez mayor. En muchas webs ya ronda el 25-30% con tendencia al alza. Esto lo deberás tener muy en cuenta.

Por suerte, hacer diseños "responsive" (adaptables a diferentes tipos de pantalla) básicos no es muy difícil. Esta misma página que es muy simple

Para ello, lo unico que deberás hacer es estructurar bien las páginas y usar dimensiones relativas (%) en las divisiones. Esto ya te permitirá hacer páginas con un diseño responsive básico.
